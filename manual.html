PyAutoMail

PyAutoMail is a tool developed by DevAlek to extract information from a spreadsheet (.tsv file) and put it into an email template to be automatically sent to all emails found in the spreadsheet.

Commom error

Sometimes the login will fail even with all the credentials being right. This happens because some email providers will block this application for security reasons. To solve this on Gmail, turn this on: https://myaccount.google.com/lesssecureapps. But be careful! Because other unknown applications will be able to log in as well!

How to use the features

Here you will see how to use the features this tool can offer you.

Word replacer

The word replacer will take a keyword from the spreadsheet and put the value in the text instead of the keywords between curly brackets.

Check TSV DataSlots

This option will show you how the program understood your spreadsheet. So, you can look for errors even before the sending!

Check Replace

This option will show you how the text will be sent to the first person on the list (topdown order).
